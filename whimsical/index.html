Some samples from Josh Comeau's course. (I haven't included everything.)

Whimsical Animations from Josh Comeau's course
  • starry-night spawn stars at random positions on button click - basic.
  • shimmer-particle button - the button shimmers - the effect does look like an obvious particle animation but it is.
  • falling-emoji-1 - hardcoded emoji elements fall from the top of the screen.
  • falling-emoji-2 - dynamically generated emojis fall from the top of the screen - my solution.
  • falling-emoji-3 - dynamically generated emojis fall from the top of the screen - Josh Comeau's particle solution.
  • oscillate-pendulum - a pendulum of 3 balls oscillates, using keyframes with custom properties & calc(). Also with play/pause button.
  • slider-rotate - rotation amount controlled by a slider.
  • bouncing-balls - bouncing balls with varying heights and durations - CSS only.
  • Versions of heart (like) button that spawns particles on click:
    • heart-particles-btn-1 - using `fromCenter` animation - Cartesian coordinates
    • heart-particles-btn-2 - using `disperse` animation with custom properties - Cartesian coordinates
    • heart-particles-btn-3 - using `disperse` animation with custom properties in CSS cos() & sin() for Polar conversion (because DOM is Cartesian) - Polar coordinates
    • heart-particles-btn-4 - convert Polar to Cartesian in JavaScript - Polar coordinates
    • heart-particles-btn-5 - with regular distribution of angle (around 360•) and distance (based on odd/even) - using CSS custom properties for Polar conversion
    • heart-particles-btn-6 - pleasingly random because a random JITTER is added to the regular distribution of angle around 360• - using CSS custom properties for Polar conversion
  • magic-wand - emits start particles on click from mouse/pointer position
  • rocketship-1 - using custom properties in CSS cos() & sin() for Polar conversion
  • rocketship-2 - convert Polar to Cartesian in JavaScript (& some other differences)
