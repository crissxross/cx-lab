VoiceOver

  1. Start screen reader, press: Command + F5
  2. Open VoiceOver rotor: Caps Lock + U
  3. Use ← → arrow keys to find Landmarks menu
  4. Select ↓↑ main & press: Enter
  5. Read all: Caps Lock + A
